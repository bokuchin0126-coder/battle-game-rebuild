# Pokemon Battle Game

javaScriptで作ったターン制バトルゲーム

## Play Game
https://bokuchin0126-coder.github.io/battle-game-rebuild/

## Demo

![game screen](screenshot.png)

## 機能
・ランダムポケモン生成（PokeAPI）
・ターン制バトル
・GameOver / GameClear
・レベルアップシステム
・ボス戦（10 / 20 / 30体目）
・HP管理
・アイテム（回復ポーション / 力の豆）
・ログ表示

## 使用技術
・HTML
・CSS
・JavaScript
・PokeAPI

## ゲームルール
・敵を倒すと経験値を獲得
・レベルアップでHPを攻撃力が上昇
・一定撃破数でアイテム取得
・30体倒すとゲームクリア
